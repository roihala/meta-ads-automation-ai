Design System · v1.0 · Dark Mode

Warm
Industrial Editorial.

A design language for aiweon — an AI agent platform built for Israeli SMBs. It pairs editorial typography with warm Mediterranean accents on a deep zinc canvas. The result is a working dark mode that feels considered rather than clinical: amber where it matters, sage where it calms, ink-black where it grounds. This document codifies the existing tokens and extends them into the patterns the product actually needs.

Iconography

Sidebar Navigation

The rail sits on the deepest surface (#09090B) and never competes with content. Active items use the active-sidebar tone — not the accent — to keep amber reserved for primary actions.

Anatomy

  • Background: --sidebar (#09090B) — the deepest surface in the system.
  • Width: 240–264px on desktop. Collapses to icon-rail at 64px below 768px.
  • Active item: --sidebar-active tone + amber icon. No bold weight change.
  • Hover item: --sidebar-hover, text rises to secondary.
  • Group labels use mono caps at 10/11px with 0.14em tracking.

Anatomy notes

Three pills, never one bar

Each pill is its own surface with its own border. The gaps between them are part of the design — they let the canvas breathe through.

Outlined active, never filled

Active nav: rgba(amber, .4) border + .06 background tint + amber icon. Solid amber would compete with primary CTAs.

Mirrors cleanly

In RTL the whole bar flips; only the brand-pill's inner divider switches sides. Icons stay (they're symbolic, not directional).

Mirroring rules

  • Set dir="rtl" at the boundary, never the leaf.
  • Borders swap sides (border-left → border-right).
  • Send/forward icons get scaleX(-1); status dots, search, settings stay put.
  • Mixed content (a Hebrew sentence with a Latin token) needs unicode-bidi: isolate on the token — already applied to .mono.
  • Numbers and codes always stay LTR — phone, IDs, URLs.

Font usage rules

Hebrew swap is automatic

Any element with dir="rtl" or lang="he" inherits the Hebrew family stack — no manual override needed.

Keep weights, swap family

Visual rhythm comes from weight + tracking. Heebo at 600 sits where Outfit at 600 sits; never re-tune sizes per locale.

Latin tokens inside Hebrew text

URLs, IDs, prices stay LTR with unicode-bidi: isolate. Apply .mono or wrap in <span dir="ltr">.
